Cyber Security Notes

1. Introduction to Cyber Security

Cyber Security is the practice of protecting computers, networks, programs, and data from digital attacks, unauthorized access, and damage.

2. Importance of Cyber Security

  • Protects sensitive data
  • Prevents financial loss
  • Ensures privacy
  • Maintains system availability
  • Prevents cyber crimes

3. Types of Cyber Threats

  • Malware: Malicious software (virus, worm, trojan)
  • Phishing: Fake messages to steal personal data
  • Ransomware: Locks data and demands payment
  • Spyware: Secretly monitors user activity
  • Denial of Service (DoS): Makes services unavailable

4. Malware Types

  • Virus: Attaches to files and spreads
  • Worm: Self-replicates without user action
  • Trojan Horse: Appears legitimate but is malicious
  • Adware: Displays unwanted advertisements

5. Cyber Security Principles (CIA Triad)

Principle Description
Confidentiality Protecting data from unauthorized access
Integrity Ensuring data is accurate and unchanged
Availability Ensuring systems are accessible when needed

6. Security Attacks

  • Passive Attack: Data is monitored but not changed
  • Active Attack: Data is modified or destroyed

7. Security Mechanisms

  • Firewall: Filters network traffic
  • Antivirus: Detects and removes malware
  • Encryption: Converts data into unreadable form
  • Authentication: Verifies user identity
  • Authorization: Grants access permissions

8. Password Security

  • Use strong passwords
  • Combine letters, numbers, and symbols
  • Avoid common words
  • Do not share passwords
  • Change passwords regularly

9. Cyber Laws and Ethics

  • Cyber Law: Laws governing cyber activities
  • Hacking: Unauthorized access to systems
  • Ethical Hacking: Legal testing of system security
  • Cyber Crime: Crimes committed using computers

10. Real-Life Examples of Cyber Attacks

  • Email phishing scams
  • ATM card skimming
  • Social media account hacking
  • Website defacement

11. Cyber Security Best Practices

  • Install software updates
  • Use antivirus and firewall
  • Backup important data
  • Avoid suspicious links
  • Use secure networks

12. Exam Tips

  • Memorize CIA Triad
  • Know types of malware and attacks
  • Understand difference between hacking and ethical hacking
  • Learn security tools and their functions
