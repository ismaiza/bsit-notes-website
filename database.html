Database Systems

Introduction

A database is an organized collection of related data. A DBMS is software used to create and manage databases.

Advantages of DBMS

  • Reduces data redundancy
  • Improves data security
  • Supports multi-user access

Types of DBMS

Hierarchical, Network, Relational, Object-Oriented
