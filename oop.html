Object-Oriented Programming (OOP)

Introduction

Object-Oriented Programming is a programming paradigm based on the concept of objects. Objects contain data and methods that operate on the data.

Core Principles of OOP

. Encapsulation

Encapsulation is the bundling of data and methods that operate on that data into a single unit called a class.

. Abstraction

Abstraction hides implementation details and shows only essential features.

. Inheritance

Inheritance allows a class to acquire properties and methods of another class.

a. Types of Inheritance

  • single inheritance
  • multiple inheritance
  • hybrid inheritance

    a. Static polymorphism

    : its achieve through method overloading and operator overloading method overloading involves defining multiple methods with the same name but different parameter lists within the same class

    b. Dynamic Polymorphism

    : also know as runtime polymorphism is achieve through method overiding and interface implementation.

    1. if Statement

    The if statement is used to execute a block of code when a condition is true. 

    if (condition) {
    // code executes if condition is true
}

    Example:

    int age = 20;

if (age >= 18) {
    System.out.println("You are eligible to vote");
}

    2. if – else Statement

    The if–else statement executes one block if the condition is true and another block if it is false.

    Example:

    int number = 5;

if (number % 2 == 0) {
    System.out.println("Even number");
} else {
    System.out.println("Odd number");
}

    3. while Loop

    The while loop executes a block of code repeatedly as long as the condition is true. 

    while (condition) {
    // repeated code
}

    Example:

    int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}

    Note: The condition is checked before the loop executes.

    4. do–while Loop

    The do–while loop executes the code at least once, then checks the condition. 

    do {
    // code executes
} while (condition);

    Example:

    int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 5);

    Difference: do–while runs at least once even if condition is false.

    5. switch Statement

    The switch statement is used to select one block from many options.

    Example:

    int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Invalid day");
}

    6. Loops in OOP

    Loops are used to execute a block of code multiple times. Common loops include:

    • for loop
    • while loop
    • do–while loop

    for Loop Example:

    for (int i = 1; i <= 5; i++) {
    System.out.println(i);
