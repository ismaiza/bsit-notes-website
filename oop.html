Object Oriented Programming (OOP)

Object Oriented Programming is a programming approach that uses objects and classes. Programs often require decision making and repetition, which is done using control structures.

. Key Features of OOP

  • Encapsulation
  • Abstraction
  • Inheritance
  • Polymorphism

    • . Class

    A class is a blueprint or template for creating objects it defines:

  • Variables(data members)
  • Methods(function)

    • Example of Class

            class Student{
             String name;
             int age;
             void study(){
                  System.out.println("Student is studying");
        }
     }

    Object

    An object is an instance of a class

    Encapsulation

    Encapsulation means wrapping data and methods together in a single unit(class)and protecting data from direct access

    Abstraction

    Abstraction means showing only essential details and hiding internal implementation

    Inheritance

    Inheritance allows one class to reuse properties and methods of another class

    Polymorphism

    Polymorphism means many forms.The same methods name behaves differently

    Example

            Class Calculator {
             int add(int a, int b){
                 return a + b;
        }
        int add(int a, int b, int c){
             return a + b + c;
        }
      }

    Control Structures in OOP

    . if Statement

    The if statement is used to execute a block of code when a condition is true. 

        if (condition) {
    // code executes if condition is true
 }

    Example:

       int age = 20;

   if (age >= 18) {
    System.out.println("You are eligible to vote");
}

    . if – else Statement

    The if–else statement executes one block if the condition is true and another block if it is false.

    Example:

       int number = 5;

   if (number % 2 == 0) {
    System.out.println("Even number");
} else {
    System.out.println("Odd number");
}

    . while Loop

    The while loop executes a block of code repeatedly as long as the condition is true. 

      while (condition) {
    // repeated code
}

    Example:

    int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}

    Note: The condition is checked before the loop executes.

    . do–while Loop

    The do–while loop executes the code at least once, then checks the condition. 

    do {
    // code executes
} while (condition);

    Example:

    int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 5);

    Difference: do–while runs at least once even if condition is false.

    . switch Statement

    The switch statement is used to select one block from many options.

    Example:

    int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Invalid day");
}

    . Loops in OOP

    Loops are used to execute a block of code multiple times. Common loops include:

    • for loop
    • while loop
    • do–while loop

    for Loop Example:

    for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

    Advantage of OOP

    • Code reusabily
    • Easy maintenance
    • Easy debugging
    • Better security

    Disadvantage of OOP

    • More memory usage
    • slower than procedural programming
    • complex for small programs
