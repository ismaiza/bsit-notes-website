Web Development Notes

1. Introduction

Web Development is the process of creating websites and web applications that run on the internet. It involves designing, building, and maintaining web pages using various technologies.

2. Types of Web Development

  • Frontend Development: Deals with the visual part of the website that users interact with (HTML, CSS, JavaScript).
  • Backend Development: Handles server-side logic, databases, and application functionality (PHP, Python, Node.js, Java).
  • Full Stack Development: Combination of frontend and backend development.

3. HTML (HyperText Markup Language)

  • Used to structure web content.
  • Basic tags:
    • <h1> to <h6> – Headings
    • <p> – Paragraph
    • <a> – Link
    • <img> – Image
    • <ul>, <ol>, <li> – Lists
  • Example:
<h1>My Website</h1>
<p>Welcome to my website!</p>
<a href="https://example.com">Visit Example</a>

4. CSS (Cascading Style Sheets)

  • Used to style HTML elements.
  • Selectors: tag, class, id
  • Example:
p {
    color: blue;
    font-size: 16px;
}

.button {
    background-color: green;
    color: white;
    padding: 10px;
}

5. JavaScript

  • Adds interactivity to web pages.
  • Common uses: form validation, dynamic content, animations.
  • Example:
function greet() {
    alert("Hello, welcome!");
}

6. Responsive Design

  • Makes websites work on different devices (desktop, tablet, mobile).
  • Techniques:
    • Media Queries in CSS
    • Flexible grids and layouts
    • Responsive images
  • Example:
@media (max-width: 600px) {
    body {
        background-color: lightgray;
    }
}

7. Web Hosting & Deployment

  • Web Hosting: Service that stores your website files on a server.
  • Domain Name: Human-readable website address (example.com).
  • Deployment: Publishing website online using FTP, Git, or hosting platforms like GitHub Pages, Netlify, or Vercel.

8. Important Terms

  • Frontend: Client-side of a website.
  • Backend: Server-side of a website.
  • Full Stack: Combination of frontend + backend development.
  • DOM: Document Object Model representing HTML elements.
  • API: Application Programming Interface for communication between systems.
