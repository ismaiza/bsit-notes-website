Networking Basics Notes

1. Introduction to Computer Networking

A computer network is a collection of interconnected devices that communicate and share resources such as data, printers, and internet connections.

2. Advantages of Networking

  • Resource sharing (files, printers, internet)
  • Fast communication
  • Centralized data management
  • Cost reduction

3. Types of Networks

  • LAN (Local Area Network): Covers a small area like a school or office.
  • MAN (Metropolitan Area Network): Covers a city.
  • WAN (Wide Area Network): Covers large areas like countries (Internet).
  • PAN (Personal Area Network): Very small area (Bluetooth).

4. Network Topologies

  • Bus Topology: All devices connected to a single cable.
  • Star Topology: All devices connected to a central hub/switch.
  • Ring Topology: Devices connected in a circular form.
  • Mesh Topology: Every device connected to every other device.

5. Network Devices

  • Router: Connects different networks.
  • Switch: Connects devices within a LAN.
  • Hub: Broadcasts data to all devices.
  • Modem: Converts digital signals to analog signals.
  • Access Point: Provides wireless connectivity.

6. Transmission Media

6.1 Guided Media (Wired)

  • Twisted Pair Cable
  • Coaxial Cable
  • Fiber Optic Cable

6.2 Unguided Media (Wireless)

  • Radio Waves
  • Microwaves
  • Infrared

7. Network Models

7.1 OSI Model

  1. Physical
  2. Data Link
  3. Network
  4. Transport
  5. Session
  6. Presentation
  7. Application

7.2 TCP/IP Model

  • Application Layer
  • Transport Layer
  • Internet Layer
  • Network Access Layer

8. Network Protocols

  • HTTP/HTTPS: Used for web communication.
  • FTP: File transfer protocol.
  • SMTP: Sending emails.
  • POP3 / IMAP: Receiving emails.
  • TCP: Reliable data transmission.
  • UDP: Fast but unreliable transmission.

9. IP Addressing

  • IP Address: Unique identifier of a device.
  • IPv4: 32-bit address (e.g. 192.168.1.1).
  • IPv6: 128-bit address.
  • Private IP: Used inside networks.
  • Public IP: Used on the internet.

10. Client-Server vs Peer-to-Peer

Client-Server Peer-to-Peer
Central server No central server
More secure Less secure
Used in large networks Used in small networks

11. Important Networking Terms

  • Bandwidth: Data transfer capacity.
  • Latency: Delay in data transmission.
  • Packet: Small unit of data.
  • Firewall: Network security system.
